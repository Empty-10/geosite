## Competitive positioning (summary)

The category splits into deep-GEO tools with no SEO (Profound, Peec, Otterly) and SEO
suites with shallow GEO (Semrush, Ahrefs). We occupy the seam and win on the **closed
loop** everyone leaves open:

1. **Attribution** — cited → AI referral traffic → conversion (GA4 + server logs).
   The field's biggest unmet need: AI mentions don't reliably correlate with revenue,
   so proving ROI is the moat.
2. **AI crawler-log analytics** — when AI bots actually crawl, what they read, errors.
3. **Fixes, not findings** — generate schema/`llms.txt`, front-load rewrites, template
   already-cited page structures onto money pages.

Plus: multi-account + white-label from day one (Profound can't), a one-time SMB audit
SKU, and hybrid pricing. And the confidence-labelling honesty none of them offer.
